Project 2

Creative Coding with p5

This p5.js sketch creates an interactive canvas with four colored quadrants, each containing bouncing circles. Circles are generated by clicking a quadrant, and the canvas can be reset with the spacebar. The project showcases randomness, motion, and user interaction.

This p5.js sketch creates a dynamic, symmetrical design where ellipses are drawn based on the mouse position. You can adjust the symmetry by pressing the '1' or '2' keys to toggle between 6-fold or 12-fold symmetry. The canvas resets to white when clicked. This project demonstrates interactive, mouse-driven art with adjustable geometric patterns.

Project 3

Problem Solving with Python

This Python script is your movie buddy, using the TMDB API to recommend films based on the genres you love. Just enter a few genres, and it’ll fetch the top 6 trending movies with titles, release dates, and ratings. If anything goes wrong, it’ll let you know, and you can exit anytime by typing "exit." A fun way to explore new movies and dive into the power of APIs.
